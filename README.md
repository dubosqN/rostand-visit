# Rostand-visit YANIS Adala / DUBOSQ Noé
Développement d'une application pour les portes ouvertes du Lycée Jean Rostand dans le cadre de la matière PPE.

Rostand-visit est une application developpée sous android (Java) communiquant avec une base de données MySql à l'aide d'une structure API de type JSON.
L'application communique avec les les fichiers php à l'aide de la libraire android Volley.

Les mots de passes sont cryptés, il est donc impossible de rajouter un utilisateur manuellement depuis la base de données.

Set up :
- Créer la base de données
- Uploader les fichiers php sur votre serveur web
- Editer le fichier de configuration des URL des fichiers d'API (ConfigUrl.java)
- Run le projet
